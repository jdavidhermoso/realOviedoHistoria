/* Real Oviedo : Historia */

Real Oviedo: Historia is an app related to Real Oviedo CF (A Spanish football club).

This app will take the info of http://www.realoviedo.info - Real Oviedo: Su historia en números. Owner: Eduardo Muñoz.

This app is gonna be built with HTML5, CSS3 & JS, and it will be compiled to Android with phonegap.
